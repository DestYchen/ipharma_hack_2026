ФОРМУЛЫ (используй, если соответствующих атрибутов нет, но есть исходные данные для расчета):

1) Константа скорости элиминации (k_el, λ_z):
   k_el = (ln(C1) - ln(C2)) / (t2 - t1)
   где C1, C2 — концентрации на терминальном участке; t1, t2 — соответствующие моменты времени.

2) Площадь под кривой до бесконечности:
   AUC0-inf = AUC0-t + C_last / k_el
   где C_last — последняя измеренная концентрация.

3) Расчет числа завершивших исследование для 2×2 кроссовер-дизайна:
   N = 2 * (Zα + Zβ)^2 * CV^2 / (ln(1.25) - ln(θ))^2
   Упрощенно (при θ = 1.0, α = 0.05, мощность 80%):
   N ≈ 315 * CV^2
   (CV в долях, например 0.30 для 30%)

4) Для 3-периодного репликативного дизайна (частично повторный):
   N_3-way ≈ (0.7–0.8) * N_2x2
   (используй как приближение, если нет точного расчета)

5) Для 4-периодного репликативного дизайна (полный повторный):
   N_4-way ≈ (0.6–0.7) * N_2x2
   (используй как приближение, если нет точного расчета)
6) Для параллельного дизайна:
   n (в одной группе) = (Zα + Zβ)^2 * CV_inter^2 / (ln(1.25) - ln(θ))^2
   Общее число завершивших:
   N_total = 2 * n
   Если CV_inter отсутствует, можно ориентировочно принять:
   CV_inter ≈ (1.5–2.0) * CV_intra
   (обязательно помечай это как допущение)
